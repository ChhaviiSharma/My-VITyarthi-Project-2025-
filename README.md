TITLE OF THE PROJECT - Banking And Money Management System

OVERVIEW OF THE PROJECT - 
It tells about the loan system of the banking sector . How can a person apply for loan and criteria to be fulfilled to be eligible for taking loan . Documents are mentioned which the customer need to submit while taking loan . The customer need to input details such as principal money , time period and rate of interest on the principal amount . At last total money to be repaid will be generated . It is also divided into monthly installments
which needs to be repaid in two years .

FEATURES - 
1. Allows user to input their loan amount, rate of interest, time period of loan taken.
2. Displays user information with the help of account number .
3. Displays amount in the bank account of the user and tells about the loan status.
4. Tells whether the user is eligible to take loan or not .
5. Allows user to customise their loan needs .
6. Calculates simple interest from the user input information. 
7. Divides the total amount to be repaid into monthly installments.

TECHNOLOGIES/TOOLS USED -
1.Dictionary is used to store the details about the user . Which can be printed with the help of account number .
2.Input system is used to take inputs from the user which make the system more practical .
3.If - elif -else loop system is used to divide users into the eligibility criteria of taking loan .
4.Nested loops are used further to further make the code understandable.

STEPS TO INSTALL AND RUN THE PROJECT -
1.Install python and VScode so that you can run the python code.
2.Firstly run the code and then enter the account number.
3.Then tell about loan status .
4.If you are eligible for applying for loan then enter input details.
5.Finally the interest will be calculated and amount to be repaid will be visible .
